# Island-Mode-72h-Stresstest · Island-Mode 72h Stress Test

Prüft, ob ein Werk 72 Stunden ohne Office-IT weiterproduzieren kann — entlang der vierzehn versteckten Abhängigkeiten, an denen Inselbetrieb in der Praxis scheitert (u. a. zentrales Verzeichnis, DNS/NTP, MES/ERP-Auftragsdaten, Lizenzserver, Etiketten-/Versanddruck, Virenschutz-Konsole, OEM-Support-Weg, Wiederherstellung, Telefonie/Alarmierung, Prozessdaten-Pufferung, manuelle Fahrweise, Offline-Dokumentation, Zutritt, Reservevorräte). Ergebnis: ein klares Urteil (inselfähig / eingeschränkt / Abbruch nach x Stunden), ein druckbarer Inselbetriebs-Steckbrief je Werk und eine nach Wirkungseintritt priorisierte Maßnahmenliste — als CSV exportierbar; optional mit Stillstandskosten je Stunde zur Bezifferung des Risikos in Euro.

Zweisprachig Deutsch/Englisch — Umschalter oben rechts.

**Live:** https://sabinefroemling-tech.github.io/island-mode-72h-stresstest/

Querverweis: Kap. 9 — ein sicherer Rückfall im No-Touch-Register setzt autarken Inselbetrieb voraus.
